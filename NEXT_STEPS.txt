# FFX Mod Manager - Future Roadmap

This document tracks planned features and improvements for the FFX Mod Manager, specifically focusing on enhancing the Fahrenheit Framework integration and overall user experience.

---

## 1. Dependency & Conflict Visualization (#2)
*   **Conflict Radar:** Implement a visual indicator when a selected mod shares files with other enabled mods. 
*   **Load Order Syncing:** Automatically update the `LoadAfter` array in the Fahrenheit `.manifest.json` files based on the physical load order set in the manager's GUI.
*   **Visual Overlap View:** Add a "Conflicts" tab in the Mod Details panel to list exactly which files are being overwritten and by which mods.

## 2. Advanced UI & Quality of Life (#5)
*   **Search & Filter Bar:** Add a persistent search bar at the top of the "Available Mods" list to quickly find specific mods by name or author.
*   **Mod Categorization:** Extend `modinfo.ffxmod` to include a `category` field (e.g., Texture, Script, Audio, UI). Add a dropdown to filter the list by these categories.
*   **Bulk Actions:** Implement a selection mode (checkboxes) to enable, disable, or delete multiple mods simultaneously.
*   **Drag-and-Drop:** Support dragging `.zip` or `.rar` files directly onto the GUI to trigger the import process.

## 3. Nexus Mods Integration (#6)
*   **Nexus Update Checker:** Allow users to input a Nexus Mod ID in the mod's metadata.
*   **API Integration:** Use the Nexus Mods API to check for version updates periodically.
*   **Update Notifications:** Add an "Update Available" badge/icon to mod cards when a newer version is detected on Nexus.
*   **Direct Download (Optional):** Explore the possibility of handling Nexus "Mod Manager Download" links directly.

---

## 4. Future Research (Other)
*   **Integrated String Editor:** Leverage `ffx_codec.py` to allow live editing of game text within the manager.
*   **VBF Ghosting:** A built-in VBF browser to help modders find and extract files for overriding.
*   **Fahrenheit Script Templates:** Boilerplate generation for C# script mods.
